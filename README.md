Mido Tech Gadgets and Accessories 
PROJECT OVERVIEW
Mido Tech Gadgets and Accessories is a website designed to simulate a modern online gadget store. The website allows users to browse products, interact with buttons, submit enquiries, and receive automated email responses. The project focuses on building a realistic user experience using only front-end technologies (HTML, CSS, and JavaScript), enhanced with EmailJS for email functionality without requiring a backend server.
Purpose of the Project:
The purpose of this project is to demonstrate practical front-end development skills by building a fully interactive website that simulates real-world e-commerce functionality. The project aims to:
•Create a clean and user-friendly online store interface
•Demonstrate interactive front-end functionality using JavaScript
•Implement form validation and user feedback systems
•Simulate real-world customer enquiry handling
•Apply consistent UI/UX design principles
•Design System (UI, Colours & Visual Identity). The website uses a soft pastel-themed design with a strong purple identity to create a modern and visually appealing aesthetic.
COLOUR PALETTE 
Primary Background: #c8a2c8 – soft lavender base for the entire site.
Navigation & Footer: #a56ca5 – deep purple for branding consistency 
Accent Colour: #fffdd0 – cream tone for text and button contrast
 Hover Accent: #8b458b – darker purple for interactive feedback Card Background: #fff0f5 – soft pink used for product and form cards 
Text Colour: #36454f – dark slate grey for readability. This colour system creates a consistent, soft, and modern e-commerce aesthetic.
TYPOGRAPHY
 The website uses a simple and highly readable typography system.
Font Family: Arial, Helvetica, sans-serif.
Chosen for its clarity, simplicity, and cross-device compatibility.
Ensures readability across all screen sizes, maintains a clean, modern, and minimal e-commerce look. Font sizes are structured using a clear hierarchy. Headings are larger and bold for emphasis. Body text is smaller and easy to read.  Buttons use bold text for visibility and interaction clarity
LAYOUT & DESIGN STRUCTURE: The website follows a structured and responsive layout system. Flexbox and Grid are used for alignment and responsiveness. Product display uses a card-based design system. Consistent spacing and padding create visual balance. Rounded corners (border-radius) give a soft modern feel. Box shadows add depth and separation between elements. Hover effects improve interactivity and user engagement.
RESPONSIVENESS: The website is fully responsive and adapts to different screen sizes:
Desktop: Multi-column layout for products and forms 
Tablet: Adjusted spacing and flexible grid layout 
Mobile: Single-column stacked layout for readability 
Navigation transforms into a vertical layout on smaller screens 
Images scale automatically using responsive CSS rules
FEATURES
 Home Page: Welcome section introducing the store.  Interactive “Shop Now” button with JavaScript alert
Products Page: Product cards displaying gadgets and accessories “Add to Cart” button with confirmation alerts. Image hover zoom effects. Clickable product images with lightbox preview
IMAGE LIGHTBOX:  Smooth overlay experience.  Easy close function (click outside or close button).
Enquiry Form (EmailJS Integration): Form validation for all required fields. Email format checking. Sends enquiry directly to business email. Automatically sends confirmation email to user. Prevents duplicate submissions by disabling button during send.
Contact Form: Simple and clean contact form. Input validation for name, email, and message. Success confirmation message after submission.
About Page: Interactive button with welcome alert message
TECHNOLOGIES USED 
•	HTML5 – structure and content
•	 CSS3 – styling, layout, and responsiveness 
•	JavaScript– interactivity and logic 
•	EmailJS – email handling without backend
Email System (EmailJS)
 This project uses EmailJS to simulate backend email functionality. 
How it works: User fills out enquiry form JavaScript validates input fields. 
Data is sent using EmailJS service. Business receives enquiry email. User receives automatic confirmation email. This allows the website to function as a dynamic system without server-side code.
Developer Challenges: 
Form Validation & EmailJS Integration: One of the most challenging aspects was ensuring proper form validation alongside EmailJS integration. It required careful handling of input validation, email formatting, and asynchronous JavaScript logic using promises. Maintaining design consistency
As the project grew, maintaining consistent spacing, button styling, and layout structure became challenging. This was solved by reusing CSS classes and standardising design rules across all pages. 
RESPONSIVE DESIGN ISSUES:  Adjusting layouts for different screen sizes required careful use of media queries and flexible layouts. Some sections had to be restructured to ensure usability on mobile devices.
